# Adaptive Streaming Research

## Problem

Internet speeds vary significantly.

## Solution

Adaptive Bitrate Streaming.

Netflix automatically changes video quality based on network conditions.

Examples:

Fast Connection → 4K

Medium Connection → 1080p

Slow Connection → 720p

Very Slow Connection → 480p

## Benefits

- Reduced Buffering
- Better User Experience
- Continuous Playback

## Key Learning

Adaptive streaming prioritizes uninterrupted viewing over maximum quality.
