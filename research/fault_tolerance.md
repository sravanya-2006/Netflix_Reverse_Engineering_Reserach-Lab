# Fault Tolerance Research

## Why It Matters

Failures are unavoidable in large-scale systems.

Examples:

- Server Failures
- Network Failures
- Database Failures

## Netflix Approach

- Redundancy
- Replication
- Automatic Recovery
- Distributed Infrastructure

## Chaos Engineering

Netflix created Chaos Monkey to intentionally test system resilience.

## Key Learning

Reliable systems are designed to survive failures rather than prevent every failure.
