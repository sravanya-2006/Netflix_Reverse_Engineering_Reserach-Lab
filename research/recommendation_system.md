# Recommendation System Research

## Purpose

Help users discover relevant content quickly.

## Input Signals

- Watch History
- Search History
- Ratings
- Viewing Duration
- Device Usage

## Common Techniques

### Collaborative Filtering

Recommends content liked by users with similar behavior.

### Content-Based Filtering

Recommends content similar to what a user has watched previously.

### Ranking Models

Prioritize content based on engagement probability.

## Why It Matters

Recommendation systems improve:
- User Engagement
- Watch Time
- Retention

## Key Learning

Netflix is solving a personalization problem as much as a streaming problem.
