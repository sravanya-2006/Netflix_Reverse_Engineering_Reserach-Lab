# Open Connect CDN Analysis

## Problem

Streaming video requires high bandwidth and low latency.

A centralized server architecture would cause:

- Buffering
- High latency
- Network congestion

## Solution

Netflix uses Open Connect, its own Content Delivery Network (CDN).

## Benefits

- Reduced Latency
- Faster Startup Times
- Lower Bandwidth Costs
- Improved Reliability

## Simplified Workflow

User
→ Nearest CDN Server
→ Video Stream

## Key Learning

Network architecture significantly impacts user experience.
